# 🎬 YouTube Dark Video Generator

Projeto de estudo para geração automática de vídeos para YouTube (Dark Channel).

## 🚀 Funcionalidades

- Geração de roteiro automatizada
- Conversão de roteiro em cenas
- Validação de assets
- Geração de vídeo com imagens + áudio
